# 巴菲特价值投资框架 — 交易原则

## 核心思想

以合理的价格买入优秀的企业，而非以便宜的价格买入平庸的企业。投资的本质是对企业内在价值的判断。

## 选股四大支柱

### 1. 护城河评估 (Moat Score: 0-10)

护城河是企业持续创造超额回报的能力来源。

**品牌护城河 (0-2分)**
- 2分：全球性品牌，定价权强（如苹果、茅台）
- 1分：区域/细分领域品牌，一定定价权
- 0分：无品牌认知或白牌竞争

**转换成本 (0-2分)**
- 2分：客户极难迁移（如企业软件、银行核心系统）
- 1分：迁移有一定摩擦但可行
- 0分：零切换成本，高度可替代

**网络效应 (0-2分)**
- 2分：强双边网络效应（如支付平台、社交媒体）
- 1分：单边网络效应或区域性网络
- 0分：无网络效应

**成本优势 (0-2分)**
- 2分：结构性成本领先（矿产资源、专利壁垒）
- 1分：规模经济带来的成本优势
- 0分：无明显成本优势

**规模效应 (0-2分)**
- 2分：垄断/寡头，市占率>30%
- 1分：行业前五，市占率10-30%
- 0分：高度分散，市占率<10%

### 2. 安全边际 (Margin of Safety)

永远在价格低于内在价值时买入，差额即为安全边际。

**估值方法优先级：**
1. DCF（现金流折现）— 适用于现金流稳定的企业
2. PE/PB相对估值 — 对比行业均值和历史分位
3. EV/EBITDA — 适用于资本密集型行业
4. PEG — 适用于成长股

**安全边际分级：**
| 安全边际 | 评级 | 建议操作 |
|----------|------|----------|
| >30% | A（极度低估）| 重仓建仓 |
| 15-30% | B（明显低估）| 正常建仓 |
| 5-15% | C（轻度低估）| 小仓位试探 |
| -5%~5% | D（合理估值）| 持有/观望 |
| -15%~-5% | E（轻度高估）| 减仓 |
| <-15% | F（明显高估）| 清仓 |

### 3. 财务质量检查

**必须满足的硬指标（不满足则一票否决）：**
- ROE连续3年 > 15%
- 资产负债率 < 70%（金融行业除外）
- 经营现金流为正且覆盖净利润 > 80%
- 无重大财务造假/审计风险

**加分项：**
- 毛利率 > 40%（高壁垒标志）
- 自由现金流持续增长
- 营收增速 > 15%且加速
- 分红率稳定 > 30%

### 4. 管理层评估

- 是否有长期主义思维（看资本配置历史）
- 内部人持股比例（>5%为佳）
- 是否有过度扩张/多元化倾向
- 信息披露是否诚实透明
- 是否有回购/增持记录

## 交易纪律

### 买入规则
1. 综合评分 ≥ 7.0/10 方可建仓
2. 单次建仓不超过目标仓位的1/3（分批建仓）
3. 必须有明确的安全边际（C级以上）
4. 买入前确认宏观环境不在「滞胀」象限

### 持有规则
1. 基本面未恶化则持有，不因短期波动卖出
2. 每季度复核一次基本面评分
3. 若评分从 ≥7 降至 <6，启动减仓观察
4. 浮盈超过50%时，评估是否目标位到达

### 卖出规则（满足任一即触发）
1. 基本面严重恶化（评分 < 5）
2. 估值到达目标位（安全边际 < -15%）
3. 发现更好的替代标的（机会成本）
4. 止损触发（保守-5%/稳健-8%/激进-12%）
5. 宏观环境切换至「滞胀」或「衰退」，非防御标的减仓

### 仓位管理
- 核心持仓（3-5只，评分≥8）：每只10-20%
- 卫星持仓（3-8只，评分7-8）：每只5-10%
- 观察仓（评分6.5-7）：每只<5%
- 总持仓标的不超过15只

## 特定市场注意事项

### 美股
- 关注美联储政策对估值的影响（利率敏感）
- 科技股侧重DCF和PEG估值
- 注意汇率波动对收益的影响

### 港股
- AH溢价是重要参考（折价>30%可能有机会）
- 关注南向资金流向
- 注意港股流动性较差的风险
- 地缘政治风险需额外折价5-10%

### A股
- 北向资金是重要风向标
- 政策面影响权重更高（+10%权重）
- 注意壳价值和并购重组的投机因素
- 关注沪深港通标的的联动效应
