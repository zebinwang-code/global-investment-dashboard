# 达里奥全天候策略 — 交易原则

## 核心思想：风险平价

不预测未来，而是为所有可能的经济环境做好准备。通过在不同经济象限中均匀分配风险（而非资金），实现全天候稳定回报。

## 四象限模型

经济环境由两个变量驱动：**增长**和**通胀**，各有预期之上/之下两种情况。

### 象限1：增长↑ 通胀↑（过热/再通胀）
- 利好：大宗商品、通胀挂钩债、新兴市场股票、黄金
- 利空：长期国债、成长股
- 操作：增配周期股、资源股，减配科技成长
- 信号：CPI加速上行、PMI>50且上升、大宗商品指数走强

### 象限2：增长↑ 通胀↓（金发女孩）
- 利好：股票（尤其成长股）、信用债、房地产
- 利空：大宗商品、黄金
- 操作：积极配置权益，加大成长股权重
- 信号：GDP增速超预期、CPI温和、企业盈利上修

### 象限3：增长↓ 通胀↑（滞胀）
- 利好：黄金、通胀保护债（TIPS）、现金
- 利空：股票、信用债
- 操作：大幅降低股票仓位，增配防御资产
- 信号：GDP放缓、CPI顽固、失业率上升

### 象限4：增长↓ 通胀↓（衰退/通缩）
- 利好：长期国债、高评级债券、美元
- 利空：股票、大宗商品、高收益债
- 操作：债券为主，股票仅保留防御板块（公用事业、必需消费）
- 信号：PMI<50且下降、收益率曲线倒挂、VIX飙升

## 风险管理原则

### 1. 仓位管理
- 单一资产不超过组合风险的25%
- 相关性>0.7的资产合计不超过40%
- 保持10-30%的现金/短债作为缓冲

### 2. 再平衡规则
- 偏离目标配比超过5个百分点时触发再平衡
- 象限切换时主动调整（不等被动触发）
- 每月至少检视一次配置合理性

### 3. 极端风险防护
- VIX > 30 时启动防御模式：减仓至50%以下
- 收益率曲线倒挂超过3个月：逐步降低权益仓位
- 信用利差急剧扩大：清仓高收益债和低评级公司

### 4. 杠杆使用
- 本系统不建议使用杠杆
- 如用户坚持，最大杠杆不超过1.5x，且仅在象限2中

## 量化信号阈值

| 指标 | 看多 | 中性 | 看空 |
|------|------|------|------|
| PMI | >52 | 48-52 | <48 |
| CPI YoY变化 | <2%且稳定 | 2%-4% | >4%或加速 |
| 10Y-2Y利差 | >100bp | 0-100bp | <0 (倒挂) |
| VIX | <15 | 15-25 | >25 |
| 失业率变化 | 下降 | 稳定 | 上升>0.5% |
| 信用利差 | <300bp | 300-500bp | >500bp |

## 象限判断算法

```
growth_score = weighted_avg(
    PMI_signal * 0.3,
    GDP_surprise * 0.25,
    employment_signal * 0.25,
    earnings_revision * 0.2
)

inflation_score = weighted_avg(
    CPI_trend * 0.35,
    PPI_trend * 0.2,
    commodity_momentum * 0.25,
    wage_growth * 0.2
)

if growth_score > 0 and inflation_score > 0: quadrant = "过热"
if growth_score > 0 and inflation_score <= 0: quadrant = "金发女孩"
if growth_score <= 0 and inflation_score > 0: quadrant = "滞胀"
if growth_score <= 0 and inflation_score <= 0: quadrant = "衰退"
```
